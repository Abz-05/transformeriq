# TransformerIQ - AI-Powered Transformer Diagnostics

## Overview

TransformerIQ is a high-performance diagnostic tool for electrical transformers using Frequency Response Analysis (FRA). It provides real-time insights and AI-powered recommendations to ensure grid reliability.

## Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Run locally
python app.py

# Deploy to Google Cloud Run
gcloud run deploy transformeriq --source . --region us-central1 --allow-unauthenticated
```
